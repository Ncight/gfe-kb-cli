# Liu2006R78 — 时域三维粘弹性人工边界 (3D VSAB)

> **一句话定位**：把无限地基对外行散射波的**吸收**（辐射阻尼）与**弹性恢复**，用截断边界上分布的**弹簧 + 阻尼器**（外加可略去的集中质量）等效，推导三维**法向/切向**时域公式与有限元落地参数。本库 [[粘弹性人工边界]] 概念的奠基源。

## 研究对象
近场波动 / 动力土-结构相互作用 (SSI) 的人工边界 (ABC) 问题。综述区分 global ABC（精度高但时空耦合、大模型不可行）与 local ABC（时空解耦）。指出 Lysmer-Kulemeyer (1969) **粘性边界 (viscous boundary)** 只吸收散射波能量、不提供弹性恢复——力学模型如悬空隔离体，低频下整个模型可像刚体一样漂移；且一维波动理论推出的边界用到多维会有大误差。故需发展三维粘弹性边界。

## 模型 / 推导
- **法向（P 波，球对称）**：由球面膨胀波波动方程推出应力-位移关系
  $$\sigma + \tfrac{R}{c_p}\dot\sigma = -\tfrac{4G}{R}\big[u + \tfrac{R}{c_p}\dot u + \tfrac{\rho R^2}{4G}\ddot u\big]$$
  $t\to\infty$ 退化为静力边界 $\sigma=-\tfrac{4G}{R}u$。等效物理元件 $K=\tfrac{4G}{R},\;C=\rho c_p,\;M=\rho R$。
- **切向（S 波）**：$\tau=-\tfrac{2G}{R}u-\rho c_s\dot u$，对应 $K=\tfrac{2G}{R},\;C=\rho c_s$（无质量）。
- **数值实现**：法向集中质量 $M$ 接在阻尼器上是不稳定系统，建模不便 → **丢弃 $M$、阻尼器外端固定**，只剩**弹簧 + 阻尼器 = VSAB**。节点参数 = 单位面积值 × 该节点周围单元面积 $A$；每节点 3 组弹簧-阻尼（2 切向 + 1 法向）。
- **修正系数**：半空间 / 低频 / 静力分量下纯理论参数不适用 → $K_1=K_2=\tfrac{\alpha_T G}{R}A,\;K_3=\tfrac{\alpha_N G}{R}A$。推荐 **$\alpha_T=2/3,\;\alpha_N=4/3$**（范围 $\alpha_T\,0.5\text{–}1.0$、$\alpha_N\,1.0\text{–}2.0$）。$R$ 取荷载点到边界的垂直距离。

## 核心发现 / 验证
4 个经典弹性波算例——爆炸源（球对称全空间）、内源（集中力全空间）、Lamb 问题（半空间表面集中力）、扭转/弯曲（半空间力偶）——与解析/广义解及粘性边界、固定边界对比：
1. **VSAB 精度最佳**；粘性边界产生明显**位移漂移**（越近边界越严重）；固定边界在荷载频率接近系统自振频率时**剧烈振荡**。
2. 爆炸源（高频膨胀波）**无需修正**即准确，与理论假设一致，验证推导正确。
3. 修正系数 $\alpha_T,\alpha_N$ **与网格尺寸、时间步及其他计算因素无关**；取不当值时 VSAB 仍优于粘性/固定边界 → 稳定性好。

## 评注（客观方法学）
- **局限（作者自述）**：仅验证**线性、均质、无限/半无限空间 + 内部激励**；分层土、非线性、地震反应分析、通用 FEM 子程序实现均留待后续。
- 作者明确：VSAB 优于常用粘性边界，但**不必优于所有 local ABC**（如廖振鹏多次透射边界 MTF）；其卖点是**概念简单 + 易嵌入通用 FEM**。
- 丢弃集中质量是工程化简化，文中称对多数情况精度影响很小。
