> [[GFE]] 软件 v2025 主用户手册：前后处理模块 PrePo 的逐功能 GUI 参考。给做土-结构动力相互作用（SSI, soil-structure interaction）/ 地下结构抗震 / 岩土地震工程的工程师，讲「在 PrePo 里怎样把一个 土-结构系统模型从导入/建几何，一路建到材料、相互作用、边界荷载、分析步、提交计算，再到后处理出层间位移角/轴压比/计算书」。

## 覆盖范围
- 前处理（第 1 章）：界面操作、模型导入导出、草图、简单几何体、几何变换/布尔/建模、网格划分、集合/表面集、材料本构、截面属性、相互作用、边界与荷载、初始条件、空间分布场、幅值函数、求解设置（分析步/输出请求/工况/作业管理器）、系统与求解器参数设置、工具、土木工程相关功能、显示、流体动力。
- 后处理（第 2 章）：界面操作、显示（变形系数/局部坐标系/拉伸显示）、云图与动画、过滤器与切割、拾取/查询/自定义场/XY 数据/阈值过滤、土木工程相关（层间位移角、层间剪力、导出场地反应结果、轴压比云图、自动生成计算书）。
- 配套对象：本手册偏 GUI 操作；底层 INP 关键字支持范围另见手册附录（与 ABAQUS 关键字对照），命令流脚本另见 [[GFE-Cmd]]。

## 关键章节

### 模型导入导出（§1.2）
GFE 作为国产土木 CAE，强调「与结构设计软件无缝对接」。导入接口：INP（GFE 计算任务文件，也可当模型文件导入）、STP/STEP、K（LS-DYNA）、GMAT（GFE 材料文件，避免重复输入参数）、YJK、PKPM、GTS NX 的 mec、Ansys CDB（APDL 命令行格式）、IFC（仅几何）、DWG（需 AutoCAD2020/2014 + GfePJ2020.arx 插件生成中间文件）。导出：几何、Key、材料、tcl（OpenSees）、INP。**YJK 接口**读 `dtlmodel.ydb`+`dtlCalc.ydb`（必需）+`Jccad_0.ydb`（可选基础），支持 YJK V4~V6；**PKPM 接口**读 `.jwd`+`_calc.jwd`+`.dwb`。导入后截面/荷载按 YJK 命名规则自动命名（v2025 为附录三，v3.4.3 已后移为附录四，详见 [[GFE-UserGuide-2026附录]]）。

### 材料本构（§1.8）——本手册对 SSI 最关键的一章
材料按「特性叠加 + 继承」组织（注：继承本构须包含被继承本构的全部必须特性）。常规类含密度、瑞利阻尼（α、β）、用户材料。
- 弹性类：线弹性、超弹性（GUI 列 Mooney-Rivlin/Ogden/Yeoh/Polynomial/Marlow/Reduced Polynomial 六种应变势能，N 通常取 1-3；附录二 *HyperElastic INP 层共 9 种——Arruda-Boyce/Neo-Hooke/Van-der-Waals 无 GUI 入口，待实测）、泡棉、粘弹性（UI 仅支持时域 Prony 级数 g_i/k_i/τ_i；频域 Tabular 需改 INP）、热传导/热膨胀。
- 塑性类：[[Drucker-Prager]] 系外的金属塑性（各向同性 / Johnson-Cook 硬化）、摩尔-库伦（Mohr-Coulomb，含摩擦角/剪胀角/黏结屈服应力）、考虑屈曲的钢筋（仅 T3D2）、蠕变（A/n/m，须配弹塑性，须开几何非线性）。
- 混凝土类：混凝土塑性损伤（CDP，含剪胀角/压缩硬化/压缩损伤/拉伸刚度/拉伸损伤）、一维非线性（梁单元，泊松比固定 0.2，可选 Johnson-Cook 率效应）。
- 岩土类：[[Davidenkov]]（仅显式动力 + 三维实体，含 A/B/γ0，可勾选「考虑液化」给 yh_m/yh_n/yh_a3/yh_c1/yh_c3 等；可由「试验数据」G/Gmax-阻尼比-应变表反算 A/B，建议拟合应变范围 1e-5~1e-3，背后基于 [[Masing法则]]）、南水模型（10 参数，综合邓肯-张+剑桥多重屈服面）、HSS（Hardening Soil-Small，基坑用，含 γ0.7/G0ref/Eur/pref）、JH-2（陶瓷玻璃硬脆材料）、E-v 与 E-B（邓肯-张族）。
- 孔隙流体：渗透性（各向同性 K + 间隙流体比重）+ 孔隙体积模量；适用 Soils/显式动力，单元 CPE3P/C3D4P。
- 预设材料：内置「一维非线性 / 混凝土塑性损伤 / 钢材」三类地下结构常用本构，一键带入参数。
- 基床系数：水平/竖向，仅供反应位移法助手算土弹簧刚度用。

### 相互作用（§1.10）——SSI 接触/界面的核心
9 类：绑定约束（Tie，主从面网格不一致但变形连续，位置容差默认 0.01）、刚体、嵌入区域（Embed，模拟钢筋/土钉加筋，舍入容差 1e-6/外部容差 0.05）、约束耦合（MPC：刚性杆/绑定/耦合）、冲击波（CONWEP 空气爆破，需 TNT 等效质量 + 单位换算系数）、接触（含[[通用接触]] general contact 与面对面接触，输摩擦系数）、弹簧/阻尼（接地弹簧 / 两点弹簧，可设弹簧刚度+阻尼系数）、连接器行为（弹性/塑性/阻尼；塑性支持 GFE HDN2 二折线、HALF CYCLE、GFE BW Bouc-Wen、GFE PEND 摩擦摆支座；阻尼 VISCOUS / GFE_DAMP2 指数型 F_d=C·v^α）、连接器（Cartesian/Align/Join 等连接类型）。
- **特殊相互作用（写入 inpx，专为地下结构抗震 SSI）**：接地弹簧（拉压异性，二维反应位移法土-结构界面）、绑定约束（法向只压不拉/切向无作用 与 法向只压不拉/切向绑定，三维结构-土界面）、绑定约束（附带材料属性，界面层）、绑定约束（三向刚度可指定，减振垫板，UI 未开发须改 inpx）。求解器支持分型号：拉压异性接地弹簧与两种只压不拉绑定支持 GFEXC/GFEXG/GFEXN；附带材料属性型仅 GFEXC/GFEXG（不支持 GFEXN）。

### 边界与荷载（§1.11）、初始条件（§1.12）
全约束、位移/转动、速度/角速度、加速度、集中力/力矩、压力、惯性力、线荷载、非结构性质量、动水压力（写入 inpx，他软件不可导入）、孔压/集中孔隙流量/表面孔隙流量、温度、体力、声压。**列车荷载（§1.11.10）**：作用在轨道几何线上（须先划网格），本质是逐节点力时程模拟移动荷载；双轨道建模，单轨力约为列车重力一半（单线简化须×2）。动轮轨力三法：直接输入、轨道实测加速度法（输轮对/转向架/车厢质量 + 一二系悬挂垂向刚度阻尼）、轨道不平顺法（输不平顺等级，详见 [[列车荷载]]）。初始条件：初始应力、初速度、初始孔压/孔隙比/温度、岩土构造二维应力。

### 求解设置（§1.15）
八种分析步：静力、静力（地应力平衡，让地应力场与荷载/边界平衡）、显式动力、隐式动力、模态、频响（模态法/直接法 + Range/Eigen frequency/Spread 间距）、模态动力（振型叠加）、反应谱（须先模态步；目前仅单方向激励 + 绝对值求和，SRSS/CQC 在后处理自定义场里做）。另有 SPH 流体动力分析步（粒子间距/计算区域/Verlet/Wendland 核）。
- **显式动力质量缩放**：迭代步长 = 质量缩放设的目标时间增量，未设默认 5e-5。振型阻尼通过指定模态结果 .db + 节点集 + 振型数 + 阻尼比设置。
- 输出请求：场输出（整体）/历史输出（追踪单点）/包络输出；子输出含节点/单元/能量/接触/统计量。
- 作业管理器（v2025 为 §1.15.4，v3.4.3 改为 §1.15.3）：创建/提交作业、写出 INP（不计算）、移除畸形单元、计算核心选 GFEXC/GFEXG/GFEXN（GFEXN 一般推荐打开几何非线性）。v3.3.0 起新增重启动计算、静力转动力、批处理参数全表，详见 [[GFE-UserGuide-2026附录]]。

### 土木工程相关功能（§1.18）——把 GFE 与通用 FEA 区分开的一章
- 创建土体：先建一维土层（自动算 P 波速/S 波速/建议网格尺寸），再生成 2D/3D 土体几何（自动建各土层集合/截面）；非均匀土体按钻孔坐标建层。
- [[粘弹性人工边界]]（§1.18.2）：把半无限空间问题转为有界有限元问题，选表面 + 形心（几何计算或直接输坐标，不支持网格）。
- 地震场地反应（§1.18.3）：从地表/基岩实测地震动 + 土层信息求各位置时程响应，三作用——给后续 SSI 时程提供地震荷载、对非线性土刚度折减加阻尼、给反应加速度/位移法提供最危险时刻土层结果分布。「用于」分等效线性化（小中震）/非线性（大震）/拟动力（反应加速度/位移法，需输结构顶底点深度）；输入位置选基岩处/基岩露头/地表。2024 版新增**调幅功能**（迭代缩放使地表 PGA 逼近目标值）。高级参数：N/MaxIter/Tol/Rr（有效应变系数）/SubLayerHeight/DampConvOrder/DampScale/UseIntgOutp/UseAmp/UseEERAMat。
- 隧道设计器、自动创建集合、施工助手（生死单元批量加减集合做施工阶段）、阻尼转换（阻尼比+频率→Rayleigh）、土体材料转换（试验数据↔[[Davidenkov]]）、结构材料&配筋转换（YJK 线性/非线性 + 配筋）、消能子结构转换（大震弹塑性下消能子结构转弹性）、隧道反应位移法助手（柔度系数 δ）、反应位移助手（自动建荷载/弹簧/相互作用，材料需加基床系数，关联 [[IPA简化抗震分析法]] 类简化法）、**选波工具 SelWave.exe**（选与目标反应谱接近的地震波，目标谱可自定义或用 GB 50011-2010《建筑抗震设计规范》规范谱（手册原文误印 GB/T50011-2010）；设防烈度/地震水准/特征周期/地震影响系数/PGA/阻尼比，平均谱比值 0.80~1.20、单条 0.50~1.50）。

### 后处理土木相关（§2.6）
层间位移角（需位移场输出 + 每层层底层顶节点集，YJK 导入自动生成 StoryDrift 前缀集合）、层间剪力、导出场地反应结果（.txt）、轴压比云图（按工况组合系数，简化轴压比需配 SF 包络场输出）、自动生成计算书（.docx，需装 Office/WPS，可不开 db）。

## 核心用法与参数
- **SSI 标准建模链**：导入 YJK/PKPM 上部结构 → 建一维土层 + 创建土体 → 材料（[[Davidenkov]] 土 + CDP/钢材结构）→ 特殊相互作用（法向只压不拉绑定约束做结构-土界面）→ [[粘弹性人工边界]] → 地震场地反应（定地震荷载 + 折减土刚度）→ 显式动力分析步（GFEXG 多 GPU）→ 提交 → 后处理层间位移角/轴压比。
- **求解器三核**：GFEXC（CPU）、GFEXG（GPU，显式动力主力，宣称比多 CPU 快 10 倍以上）、GFEXN（隐式静力/牛顿迭代，Conv_Tol 默认 0.005、Max_Iter 默认 6，支持 OOC 硬盘外存模式）。
- **显式步长默认 5e-5**；各单元稳定步长折减系数 *_DtCo（四面体 0.75、六面体/楔形/四边形 0.5、梁 0.25、三角形 0.25-0.35），越小越保守、质量放得越大。
- **求解器参数 config.txt**：界面外的隐藏参数在 `安装目录/program/config.txt` 改；查找优先级=批处理指定路径 > 结果目录 > 程序目录 > 默认。关键项：Hex_HourGlass_Co（沙漏刚度，取 0 即纯单点积分）、IsRemoved*_Co（剪切应变达阈值杀单元）、Sigma_Co/YH_CO（液化范围/程度，Davidenkov 液化用）、RenGongBianJie_Nonlinear（人工边界非线性，建议 0，科研用 1）、RenGongBianJie_OnlyRF（人工边界只取反力，用于非人工边界模型如滚轴边界的地应力平衡）、Use_Penalty_Coupling（0=拉格朗日乘子精度好但易发散，1=罚函数稳定）。
- 自动保存最小间隔 10 分钟；SPH 参数须 SI(m) 标准单位。

## 坑与注意
- **β 数值刚度阻尼默认 0**：config 里 Beta_Damp_Co 默认 0；GFE 显式动力中 β 阻尼会触发质量缩放级联 + CDP 报错（与本地工程经验一致，*Damping 里 β 必须保持 0）。
- **本构↔分析类型/单元强约束**：[[Davidenkov]]、JH-2 仅显式动力 + 三维实体；摩尔-库伦/HSS 仅二维/三维实体；超弹/泡棉/粘弹性/蠕变须打开几何非线性；C3D8I 不支持超弹/泡棉/粘弹性；孔隙流体仅 CPE3P/C3D4P。退化规则：塑性/摩尔-库伦在不支持的分析类型中会退化为线弹性。
- **特殊相互作用 + 动水压力 + 三向刚度绑定约束写在 inpx**，其他软件无法导入计算；三向刚度绑定约束 UI 未开发，须手改 inpx。
- **频域粘弹性、连接器某些类型须改 INP**；UI 粘弹性仅时域 Prony。
- 与 ABAQUS 的差异：质量缩放后 GFE 把迭代步长固定为质量缩放最小步长（ABAQUS 实际步长有时略大）；INP 须加 `*Section Controls`；INP 关键字支持是 ABAQUS 子集（详见手册附录关键字对照）；选 GFEXN 若有分析步未开几何非线性会弹窗强制全开。
- **导入版本限制**：YJK 仅 V4~V6；DWG/PKPM 仅 AutoCAD2020/2014 + GfePJ2020.arx 插件；其余版本未测试不保证。
- 列车荷载单轨简化必须把轮轨力×2 或施加两个相同荷载；反应谱目前仅单方向 + 绝对值求和，多方向/SRSS/CQC 要到后处理自定义场做。
- 轴压比「简化轴压比计算」勾选前提是动力时程工况设了 SF 的包络场输出，否则不应勾。
