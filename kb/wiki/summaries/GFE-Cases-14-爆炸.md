> 在一个半地下室的「土-结」整体模型内部最低层引爆，用 CONWEP 经验公式（空气爆破）施加冲击波荷载，做显式动力分析，演示 [[GFE]] 的爆炸荷载（冲击波）功能、显式动力分析步与单元失效（侵蚀 erosion）求解器开关，输出结构受压/受拉损伤与位移。

## 工程背景与模型

- **复用模型**：本案例不另建几何，直接复用官网案例 7「土-结模型-地震动力非线性分析」的模型文件 `GFE-model-Plastic`（即 [[地上-地下耦合结构]] 类的土-结整体模型）。在此既有模型上叠加一次内部爆炸工况。
- **结构对象**：一个半地下室结构（含 `floor0`、`floor1`、`floor2` 等分层几何集，结构几何对象名 `SuperStru`，后处理过滤用的结构集合名 `jiegou`（几何集，dump 实测）），坐落于土体中。
- **爆炸场景**：起爆点（参考点）放在结构**最低层**内部；迎爆面取**最底下两层**的内表面。属于结构内部近场爆炸响应分析。
- **单位制**（CONWEP 必须自洽，手册明确规定）：质量用**吨**、长度用**米**、压力用**千帕（kPa）**。

## 演示的 GFE 功能

- **冲击波荷载 / 爆炸荷载（blast load）**：通过「相互作用」下的「冲击波属性」+「冲击波」两件套实现，底层用 **CONWEP** 经验公式（[[CONWEP]]）计算入射/反射超压时程，定义类型选「空气爆破（air blast）」。
- **显式动力分析（explicit dynamic）**：爆炸属高速短时强冲击，用 [[显式动力分析]] 步求解（分析时间 1 s）。
- **单元失效 / 侵蚀（element erosion）**：通过修改求解器 Config 文件的 `IsRemoved` 系列开关，让损坏构件在计算中被删除，模拟构件破坏失效。
- **混凝土损伤本构**：复用原模型的混凝土塑性损伤（CDP）本构（`concrete_damaged`，dump 实测；另有土体 Mohr-Coulomb + 骨架曲线、钢材双折线塑性），输出受压损伤（DAMAGEC）/ 受拉损伤（DAMAGET）。
- **场输出与后处理**：节点位移、冲击波压力、单元应力/应变/塑性应变/损伤的提取与云图查看。
- **CPU / GPU 双求解核心**：`GFEXC`（CPU）与 `GFEXG`（NVIDIA GPU）可选。

## 关键操作与参数

- **起爆点（参考点）建立**（§14.2）：用搜索框过滤 `floor`，按 Ctrl 选中 `floor0-All`、`floor1-All` 单独显示地下一层；「查询」拿到大致坐标后用「参考点」精确键入坐标建参考点。
- **冲击波属性**（§14.3）：「相互作用」→双击「冲击波属性」→「定义」选**空气爆破**→填 CONWEP 参数。实测 IncProp-1：def=0（空气爆破），data=[100.0, 1000.0, 1.0, 1.0, 1000.0]（dump interaction_iw_prop_mgr.txt；字段与药量/距离的映射手册未明，存疑待实测）。⚠ 务必先确认建模单位为 吨-米-千帕，否则 CONWEP 超压量级会错。
- **爆炸荷载装配**（§14.4）：
  - 先建**源点节点集**（选参考点，集合类型「节点」）。
  - 再建**迎爆面表面集**：单独显示 `floor0-All`/`floor1-All`/`floor2-All`，双击「表面集」，选面方式「几何」，Shift 全选确定（最底两层内表面），确定后再选择「品红」（面方向颜色选择，决定迎爆面法向，不可省略，§14.4(2)）。
  - 双击「相互作用」→「冲击波」：源点选源点节点集，表面选迎爆面表面集（⚠ 手册 §14.4(3) 原文"选择新建立的节点集"系笔误，实为表面集——实测 IW-1.surf_name='surface_blast'），属性选 §14.3 建好的冲击波属性；**起爆时间 0 s，数量放缩系数（药量缩放）= 1**。
- **分析步**（§14.5）：显式动力分析，时长 **1 s**，其余默认。实测（ch14_plastic dump）：分析步 Dyna-1（period=1.0 s，nlgeom=True，含质量缩放 target_time=5e-05）；源点节点集 Node（单节点 154200）；迎爆面表面集 surface_blast（920 面）；工况 Case-1=[Initial, Dyna-1]。
- **两套场输出**（§14.6）：① 节点输出 = 位移 + 冲击波压力；② 单元输出 = 受压损伤 + 受拉损伤 + 应变 + 塑性应变 + 应力。
- **工况**（§14.7）：初始分析步内加边界约束；显式动力分析步内挂上述场输出。
- **求解器单元失效开关**（§14.8，关键坑）：编辑 GFE 安装目录 `Program`（实例路径 `…\GFE_2_11\program\config.txt`）下 Config 文件——把 `IsRemoved_Co`、`IsRemoved2_Co` 改为 **1**，并新增 `IsRemoved3_Co = 1`。不改这三个开关，破坏单元不会被删除，无法体现构件失效。
- **提交作业**（§14.9）：通用→作业管理器→创建，工况选好，计算核心 `GFEXC`（CPU）或 `GFEXG`（GPU），提交。
- **后处理**（§14.10）：右上「后处理」进入；过滤器选「单元」→「集合」→`jiegou`，查看爆炸工况下结构位移与受拉破坏云图（受拉损伤值 0 ~ 0.5 色阶；手册 §14.10 原文"罕遇地震下"与"图 11.25-1"系第 11 章复制残留，本案例无地震输入）。
