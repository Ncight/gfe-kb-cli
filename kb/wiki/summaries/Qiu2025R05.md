# Qiu2025R05 — SIUS 在不同地震动特性下的损伤演化

> **系列定位**：在 [[Qiu2023R03]] 建立的 SIUS 机理上，进一步研究**地震动特性（频率成分 + 入射角）**如何影响 [[SIUS]] 的损伤演化与 [[地震破坏路径]]。

## 研究对象 / 模型
同 [[Qiu2023R03]] 的 [[SIUS]]；新增**斜入射角**地震输入（[[粘弹性人工边界]] + 波动输入支持 oblique angle）。

## 核心发现
1. **脆弱位置**（同 R03）：地上底部（[[SAUI]] 强）+ 地下外侧（土-结构作用强）。损伤演化：地上 底→顶、地下 外→内。
2. **垂直入射(0°)破坏路径**（6 步）：① 地上底部梁 → ② 地下底部梁 → ③ 地下顶部外侧梁 → ④ 地上底部柱 → ⑤ 地下底部柱 → ⑥ 地上中部梁柱。**梁先于柱、地上底部最先**。
3. **频率敏感性**：**地下部分 ← 低频**（低频主导地震位移/土变形）；**地上部分 ← 高频**（高频主导结构振动惯性）。界面层 A0 受上覆土约束、由低频位移控制；地上 A1 由高频振动控制。
4. **入射角效应**：角增大 → 地上响应**减小**（水平作用变小）、地下响应**增大**（沿入射方向呈斜向分布而非对称）。30° 时破坏路径转为**地下优先**；角越大，地下外侧损伤超过地上。

## 评注
把"地震动特性"作为变量系统接入 SIUS 损伤演化。频率二分（地下↔低频位移控制 / 地上↔高频振动控制）与"斜入射放大地下"是两条物理上较一般、可推广的结论。
