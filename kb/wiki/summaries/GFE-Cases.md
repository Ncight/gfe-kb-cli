> 广州颖力《GFE 实际案例操作手册 v2025》总览。15 个由浅入深的工程算例，从单构件建模一路到土-结构动力相互作用、施工模拟、流固耦合与特殊荷载，是把 [[GFE-UserGuide]] 的功能落到完整工程流程的实操样板库。每个案例已拆为独立子页（见下）。

## 案例地图（按主题）

**建模入门 / 单构件**（几何 + 布尔 + 单元 + 模态/静力）
- [[GFE-Cases-01-球铰支座]] — 钢球铰支座；布尔裁剪造空心壳 + 合并传力，[[实体单元]] C3D4，模态 + 静力。
- [[GFE-Cases-02-钢骨混凝土]] — SRC 短柱；多材料共节点合并 + [[嵌入区域]] 把钢筋笼嵌入混凝土。
- [[GFE-Cases-03-华夫板框架]] — 带圆孔华夫板框架；[[实体单元]]/[[壳单元]]/[[梁单元]] 混合建模 + 频响分析。

**土-结构相互作用（SSI）抗震**（[[地震场地反应]] + [[粘弹性人工边界]] + [[绑定约束]]）
- [[GFE-Cases-04-核电站]] — 厂房 + 安全壳置入四层水平成层场地（对照案例 05 的起伏非均匀场地），时程 SSI；标准 SSI 功能链样板。
- [[GFE-Cases-05-非均匀场地]] — 钻孔驱动的层面起伏成层场地，案例 04 的进阶对照。
- [[GFE-Cases-10-抗震地铁站]] — [[YJK]] 导入含桩地铁站，弹性中震 → [[Davidenkov]] 弹塑性罕遇。
- [[GFE-Cases-11-抗震地上地下]] — 带地下室高层 + 大土体整体 SSI，弹性 → 弹塑性。

**岩土 / 施工模拟**（[[单元生死]] + 摩尔库伦 + 施工助手）
- [[GFE-Cases-06-基坑开挖]] — 含支护体系分六步开挖，评估对旁侧既有建筑影响。
- [[GFE-Cases-07-锚杆隧道]] — 分层软岩中开挖 → 锚杆 → 喷混循环施工。
- [[GFE-Cases-12-施工地上地下综合体]] — 逐层开挖 + 自下而上建造的施工阶段静力分析。

**流固耦合（[[SPH光滑粒子流体动力学]]）**
- [[GFE-Cases-08-SPH小球落水]] — 自由液面流固耦合入门样板。
- [[GFE-Cases-09-SPH水桶晃动]] — FEM 壳 + SPH 水的耦合晃荡，两段式工作流。

**简化抗震**
- [[GFE-Cases-13-反应位移法]] — 带地连墙地下结构拟静力抗震，反应位移助手一键生成等效荷载。

**特殊荷载 · [[显式动力分析]]**
- [[GFE-Cases-14-爆炸]] — 内部引爆，[[CONWEP]] 冲击波 + 单元失效（侵蚀）。
- [[GFE-Cases-15-列车振动]] — [[列车荷载]] 移动轮轨力，振动经隧道-土体传至结构。
