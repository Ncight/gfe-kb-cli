> 用 ETABS 反应谱分析对比 [[网格楼板]]（grid slab / 密肋楼板）与 [[网格楼板]] 在 H/T/L 三种平面不规则 G+9 中高层中的抗震表现，结论：网格楼板挠度降约 71.1%、壳应力降逾 74%，在 [[平面不规则结构]] 与 [[平面不规则结构|非对称]] 结构里更显著抑制 [[扭转不规则]]，但对称结构本身因刚度/荷载均衡而受益有限。

## 研究对象

针对城市化带来的 [[平面不规则结构]]（plan irregular structures），按 [[对称轴]] 分为三类并以钢筋混凝土 G+9（10 层、层高 3.5 m）框架代表：
- **H 形** = 双轴对称（symmetric），模型记 HCB（传统板）/ HGB（网格板）；
- **T 形** = [[平面不规则结构]]（single-axis symmetric），记 TCB / TGB；
- **L 形** = [[非对称]]（asymmetric，双轴均不对称），记 LCB / LGB。

核心问题：把 [[网格楼板]]（grid slab，又称密肋/华夫板，薄面板 + 双向肋梁）与 [[网格楼板|conventional solid slab]]（一向板/二向板）放进上述三种不规则平面，对比 [[楼板隔板|diaphragm]] 在 [[挠度]]、壳应力、[[底部剪力]]、[[模态周期]]、[[层间位移]]、[[层间位移角限值]]、[[扭转不规则]]、[[扭转隔板转角]] 上的差异。作者主张：楼板常被视为次要构件，但在不规则结构中其刚度与荷载分配作用被低估，网格楼板与平面不规则的耦合效应此前研究稀缺。

## 模型（软件/单元/参数/分析方法）

- **软件**：[[ETABS]] v20（Computers and Structures, Inc.），有限元建模 + [[反应谱法|Response Spectrum Method]] 评估地震响应（线性静力 + 动力）。
- **几何**：X、Y 双向各 5 跨 × 12 m；全部模型总楼板面积保持一致以保证可比性；楼板厚 100 mm；楼板（传统与网格）均设为 [[刚性隔板假定|rigid diaphragm]]。
- **材料/设计参数**（Table 1）：抗弯钢框架 SMRF；混凝土 M40；钢筋 Fe 500；重要性系数 1.5；[[地震分区系数]] 0.36（印度抗震 Zone V，最高烈度区）；规范主体为 [[Is1670R12|IS 1893 (Part-1) 2016]] 与 IS 456:2000、NBC 105 2020。
- **网格肋参数变量**：肋宽固定 125 mm；肋深 250 / 375 / 500 / 625 mm（即 [[深宽比d/b]] = 1～5，宽 125 mm）；肋间距 1000 / 1500 / 2000 / 3000 / 4000 mm。先以挠度（IS 456 限值 span/250 = 48 mm）与壳应力为准则做网格优化，再代入整体抗震分析。
- **荷载**（Table 2）：楼面活荷 4 kN/m²、屋面活荷 1.5 kN/m²、楼面附加恒载 1 kN/m²。
- **周期经验公式**：IS 1893 取 T_a = 0.075·h^0.75。

## 核心发现（含定量）

**楼板优化（重力荷载，§3.1–3.2）**
- 最优配置 = 肋间距 2000 mm + [[深宽比d/b]] = 5：挠度 40.13 mm（< 48 mm 限值），相比传统板 139.14 mm 降低约 71.1%。
- 3000 mm 间距 + d/b=5 挠度 43.62 mm 也合规，但不留意外荷载裕度，故选 2000 mm 更稳健。
- 壳应力：2000 mm + d/b=5 时底壳应力 5.81 MPa、顶壳应力 11.34 MPa；相对传统板（底 22.48 MPa、顶 48.41 MPa）分别降 74.2% 与 76.6%。相比 3000 mm 配置（底 5.77、顶 10.76 MPa），2000 mm 仅顶壳应力增约 5.4%、底壳应力增约 0.7%，换来更高刚度与抗意外荷载能力。

**整体抗震（反应谱，三种平面，§3.3–3.8）**
- **[[底部剪力]]**：网格板一致高于传统板（因肋梁增刚 → 周期略缩 → 地震力略增）。X 向 H/L/T 增幅 20.89% / 22.75% / 21.80%；Y 向 18.45% / 21.61% / 20.42%。绝对值 H 形最高（X 15170.42 kN，Y 15965.68 kN），对称结构底部剪力高于非对称（与 Pandian et al. 2024 一致）。
- **[[模态周期]]**（第一基本模态）：HCB 1.969 / HGB 1.945、TCB 1.963 / TGB 1.956、LCB 1.986 / LGB 1.972 s。网格板使周期降 HGB 1.22% / TGB 0.36% / LGB 0.71%。FE 周期（1.945～1.986 s）比 IS 经验公式 1.08 s 高约 80–83%，作者据此质疑规范经验公式系统性低估周期。
- **[[层间位移]]**（顶层最大）：HCB 115.96 / HGB 114.09、TCB 136.07 / TGB 124.57、LCB 143.83 / LGB 138.08 mm。网格板降顶层位移 H 1.63% / L 4.17% / T 9.23%；L 形位移最大。
- **[[层间位移角限值]]**（最大）：HCB 0.00163 / HGB 0.00153、TCB 0.00188 / TGB 0.00165、LCB 0.00198 / LGB 0.00181。L 形最高、H 形最低；网格板降幅 H 0.72% / L 3.19% / T 8.16%。
- **[[扭转不规则]]**（IS 1893 限值 1.5）：HCB/HGB 均 1.00、TCB 1.30 / TGB 1.14、LCB 1.52（超限）/ LGB 1.41（回到合规）。网格板使 L、T 形抗扭刚度分别提高约 30% 与 27%，H 形不变。
- **[[扭转隔板转角]]**：HCB/HGB = 0、TCB 0.00086 / TGB 0.000447、LCB 0.001431 / LGB 0.001225（沿高度顶层最大，X 向大于 Y 向）。对称越强转角越小。

**总结论**：网格楼板虽抬高底部剪力，但显著降挠度/壳应力/位移/位移角/扭转，且对 [[平面不规则结构]] 与 [[非对称]]（T、L 形）增益最明显，对 H 形（已均衡）增益微弱；同时对称平面整体优于非对称平面。

## 评注（方法学）

- **方法定位**：纯设计软件（ETABS）+ [[反应谱法]] 的弹性对比研究，无 [[土-结构相互作用]]（固定基座假定），无非线性时程，无 [[一维土层]] 场地反应——与本批中走精细 FE/显式动力的 SSI 论文（如 [[Zhao2024R21]]、[[Scarfone2020R10]]）属不同尺度层级。作者自己也指出弹性分析在结构进入非线性后会低估位移角。
- **网格优化的双准则**（挠度 + 壳应力）逻辑自洽，定量结果（71.1% 挠度、74%+ 应力）对密肋楼板选型有直接参考；但"总楼板面积一致"这一可比性约束的实现细节（质量是否等同）未充分说明，而底部剪力差异本质来自质量/刚度变化，结论的因果链略显粗略。
- **底部剪力上升却"抗震性能更好"** 的表述需谨慎：底部剪力升高源于刚度增加导致周期缩短、地震力增大，并非损伤指标；作者用位移/位移角/扭转作为性能改善证据，逻辑成立但应明确区分"需求"与"能力"。
- **对规范经验周期公式的批评**（低估 80–83%）值得注意，但 IS 经验式本就偏保守用于放大地震力，"低估周期"在工程上通常是安全侧，作者未充分讨论这一点。
- 与 [[Vaziri2024R37]]（L 形不规则 BRBF 钢结构 SSI）、[[Heshmat2024R17]]（不对称中高层桩筏扭转 SSI）、[[Oz2025R36]]（TMD 减 L 形扭转）同属"平面不规则 + 扭转"主题，但本文唯一聚焦点为楼板系统（网格 vs 实心）这一变量。
