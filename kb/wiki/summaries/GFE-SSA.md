> [[GFE]] 地下结构动力分析软件（GFE-SSA）技术手册：面向地铁车站、隧道等地下结构的抗震分析师，把规范（GB 50909-2014、GB/T 51336-2018）要求的三类方法——时程分析（直接法）、反应加速度法、反应位移法——集成进一套从 YJK/盈建科结构模型导入到土-结构一体化建模、求解、配筋出计算书的流水线，并用同一地铁车站算例与某国际通用软件（手册称"软件 A"）逐项对标精度与速度。

## 覆盖范围
- 第 1 章：地下结构抗震分析方法的理论基础（控制方程、[[粘弹性人工边界]]、一维场地地震反应分析、地震动输入、土体非线性[[本构模型]]、初始应力条件、反应加速度法与反应位移法原理）。
- 第 2 章：标准测试算例——某两层三跨地铁车站 + 四层场地 + E2/E3 基岩地表地震动的全部材料与几何参数。
- 第 3 章：二维时程分析完整软件操作（10 步）+ E2 线性 / E3 非线性结果 + 土层液化影响。
- 第 4 章：反应加速度法软件操作与结果对标。
- 第 5 章：反应位移法软件操作（地基弹簧 + 三类荷载）与结果对标。
- 第 6 章：三维时程分析（60 万节点 / 320 万单元规模）+ GPU 加速实测。
- 第 7 章 + 附录：精度/速度汇总结论、重力静力分析、模态分析。

## 关键章节

### 第 1 章 理论方法
- **控制方程与人工边界**：把截断边界上的总反应分解为自由场（上标 F，由场地反应分析给定）+ 散射场（上标 S，未知），散射场用[[粘弹性人工边界]]模拟，最终地震荷载是作用于边界节点的等效节点力 `f_B = K∞·u_B^F + C∞·u̇_B^F + f_B^F`。
- **[[粘弹性人工边界]]参数**：二维法向 `K_N = (1/(1+A))·(λ+2G)/(2r)`、`C_N = Bρc_p`，切向 `K_T = (1/(1+A))·G/(2r)`、`C_T = Bρc_s`；三维公式分母为 r（非 2r）。r=近场结构几何中心到该边界点**所在边界线/面**的（法向）距离，非到边界节点的距离（§1.1.2，角部两者差异可达 √2~√3 倍）；推荐 A=0.8、B=1.1。GFE 可自动施加，并考虑土层性质变化与不规则边界几何。
- **一维场地地震反应分析**：水平成层土体、平面体波竖直入射的等效线性化分析（对标 EERA / SHAKE），支持三种输入位置——地表输入、基岩输入、基岩露头（已知基岩地表运动折半在基岩输入；已知基岩运动强制于基岩；已知场地地表则反演）。与土-结构分析无缝对接，无需导入导出。
- **地震动输入**：支持 P 波 / SV 波 / SH 波及不同边界几何形状（二维、三维立方体、柱体、半球）。
- **土体非线性[[本构模型]]**：摩尔库伦（Mohr-Coulomb）与可考虑土体动力特性及液化的 [[Davidenkov]] 本构（参数 A、B、γ0；液化另加 γtv、m、n、a3、c1、c3）——SSA 手册无独立 Drucker-Prager 条目。结构侧：[[梁单元|纤维梁单元]] + [[壳单元|分层壳单元]]内置钢筋弹塑性 + 混凝土塑性损伤本构，预设 Q235-Q460、HPB235-HPB500、C30-C80 材料库。
- **初始应力条件**：非线性时程前先对人工边界施法向约束 + 施体力做静力分析，再把应力与边界约束力传给地震反应模型；GFE 把静力→动力两段做成连续统一，避免数据手动传递（自重下位移场应≈0、应力场应与自重应力理论一致）。
- **反应加速度法**：土层-结构整体模型，底固定 + 侧向水平滑移，全模型受惯性力；有效反应加速度 `a_i = (τ_i − τ_{i−1})/(ρ_i·h_i)`，取结构顶底板最大相对位移时刻的土层剪应力差分。
- **反应位移法**：荷载-结构子结构模型，结构用[[梁单元]]、土用地基弹簧；三类荷载=土层相对位移（强制于弹簧远端）+ 结构惯性力 `f_i = m_i·ü_i` + 结构周围土层剪力（矩形侧壁可简化 `τ_s = (τ_U + τ_B)/2`）。地基弹簧刚度两法：基床系数 `k = K·L·d`，或静力有限元法（去除结构的孔洞土体四侧面分别加垂直/水平力反算）。

### 第 3 章 二维时程分析（10 步操作流程）
导入 YJK ydb 模型 → 创建土层材料及几何 → 整体模型组装（平移调位 + 布尔裁剪挖洞）→ 分析步设置 → 网格划分 → 土-结构相互作用设置 → 人工边界设置 → 场地地震反应分析与地震动输入 → 场输出设置 → 设工况建任务提交。结构构件用[[梁单元]]，土用平面应变单元，接触用 Tie 绑定约束、地连墙用 Embed 嵌入约束。

### 第 4/5 章 简化法操作
- 反应加速度法：ERA 窗口 `Use for` 选 **Pseudo Dyna/拟动力分析**，填结构顶底深度（算例 2–17.43），Compute 后导出"结构顶底位移差最大时刻"的加速度沿深度曲线 → 升序排列 → 建 `Amp-AX` 幅值函数 → 以 `Gravity/惯性力` 类型按 Y 向空间分布施加。边界：底面 Encastre 全约束、左右侧面仅约束竖向 U2=0。
- 反应位移法：同法导出加速度 `Amp-AX`、相对位移 `Amp-UX`（须减结构最低点位移）、剪应力 `Amp-SX`；地连墙底部两点（Set-base）约束 U1/U2（不是约束整个底板或底面，5.1.2(a)）；接触界面加地基弹簧；用 Line Load 施加相对位移（分量填弹簧刚度）与剪应力。

### 第 6 章 三维时程分析
操作与二维一致（导入选 3D、查找接触选"几何面"、裁剪可用 ydb 自带外轮廓实体）。算例模型 X×Y×Z=500×200×50 m，602126 节点 / 3209579 单元（梁 2736 + 壳 44813 + 实体 3162030）；土用一阶四面体 3 m 网格，楼板侧墙用壳、梁柱用梁、1.0 m 网格。

## 核心用法与参数
- **算例标准参数**（可作建模缺省参考）：两层三跨车站，层高 7 m、埋深 4 m，外墙 700 mm / 底板 900 mm / 中板 400 mm / 顶板 750 mm，中柱 700×1000 mm @7500 mm，中柱 C50、其余 C35。场地四层：粉质黏土素填土（5 m）→ 粉质黏土（20 m）→ 粉质黏土（16 m）→ 微风化石灰石（9 m）。
- **[[粘弹性人工边界]]推荐值**：A=0.8、B=1.1。
- **Davidenkov 土层参数**（算例三层）：A≈1.04–1.09、B≈0.4–0.46、γ0≈0.00036–0.00041；液化参数 γtv=0.001、m=0.345、n=6.689、a3=0.45、c1=1.051、c3=1.25。
- **分析步**：二维/三维动力时程必须选 **动力显式（Dynamic Explicit）**；非线性时程在动力步前要加一个整体重力静力步做地应力平衡。
- **导入差异**：非线性分析导入 ydb 时选 YJK 配筋或默认配筋，结构材料选"塑性"；E3 地震要在 ERA 的"地震水准"勾选 E3。
- **场地输入位置**三选一：地表 / 基岩 / 基岩露头（Outcrop）。
- **求解架构**：CPU+GPU 异构并行显式动力，宣称比多 CPU 并行快 10 倍以上（2000 万自由度模型 30–50 倍）。

## 坑与注意
- **黏弹性 vs 粘弹性命名**：手册原文写"黏弹性人工边界"，库内概念页为 [[粘弹性人工边界]]，同一概念，链接以库内页名为准。
- **二维/三维边界公式分母不同**：二维 K 用 2r，三维用 r，套公式别用错维度。
- **反应加速度/位移法取值时刻统一**：加速度、相对位移、剪力三类荷载都取"结构顶底板相对位移最大时刻"的同一时刻分布，不可各取各的峰值。
- **反应位移法相对位移要减基准**：`Amp-UX` 必须减去结构最低点处位移（相对量），直接用绝对位移会错。
- **简化法 vs 时程的固有误差**：手册自评 E2 下反应加速度法、反应位移法相对时程法误差约 7.36% / 7.72%，属正常范围而非软件问题；时程法二维/三维互差仅 0.80%。
- **显式动力 + 阻尼**：动力显式求解对阻尼/质量缩放敏感（参见 [[GFE-Explicit]] 显式专版），与本手册"动力显式"分析步配合时注意稳定性。
- **与 ABAQUS 等通用软件的差异**：GFE 把"场地反应分析→等效地震荷载→人工边界节点力"和"静力地应力→动力"两条数据链做成内部连续过程，无需手动导入导出、不需自写 *Amplitude / 子程序施加边界力；术语对标 ABAQUS（Encastre、Tie、Embed、Gravity、Line Load、U1/U2、Field/History Output），但入口在 ERA/Art BCs 等 GFE 专属模块。
- **手册局限**：仅给一个地铁车站算例；E3 非线性结果无"软件 A"对照（软件 A 只跑了 E2 线性）；图表多为云图截图，定量对比集中在差异率文字（如剪力 481 vs 483 kN，差 0.41%；E2 最大层间位移角各法差异率 0.13%–1.52%）。
