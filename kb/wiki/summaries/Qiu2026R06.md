# Qiu2026R06 — SIUS 整体简化抗震分析法(IPA)与性能阈值

> **系列定位**：系列的**工程方法篇**——针对 [[SIUS]] 提出可替代耗时全时程(DTHA)的 [[IPA简化抗震分析法]]，并给出各区性能状态阈值。承接 [[Qiu2023R03]] 机理与 [[Qiu2025R05]] 响应规律。

## 动机
DTHA（全时程）计算量大、工程不便；单独地上(pushover)或单独地下(反应位移法)的简化法都**不能直接用于耦合的 SIUS**。

## 方法：[[IPA简化抗震分析法]]
**同时计入**地上部分的**惯性振动** + 周围土对地下部分的**强迫位移**，合成一套简化分析；以 DTHA 为基准验证。

## 核心发现
1. **最脆弱构件**：地上 Column 7、地下 Column 1、界面 Beam 7。
2. **IPA 精度**（对 DTHA 误差，地上/地下/界面）：PGA 0.02–0.10 g → ≤22% / 23% / 9%；0.15–0.30 g → 27% / 8% / 11%；0.40–0.80 g → 28% / 10% / 6%。软土 ≤35% / 10% / 20%，硬土 ≤25% / 6% / 15%。地下与界面精度高，地上误差略大但可接受。
3. **五级性能状态**（I 完好 → V 极限/濒临倒塌）阈值：
   - 地上 IDR：**0.2 / 0.4 / 1.0 / 2.0 %**
   - 地下 IDR：**0.1 / 0.2 / 0.55 / 1.0 %**
   - 界面 IRA：**0.1 / 0.4 / 0.9 / 1.4 %**

## 评注
把"地上 pushover 惯性 + 地下强迫位移"拼成一套适配 SIUS 的简化法，是该系列从机理走向工程实用的落点。误差以地上侧偏大(~28%)，反映地上惯性振动的简化更难捕捉。
