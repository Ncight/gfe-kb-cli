# Pinto-cruz2024R47 — 广义连续模型分析高层自由振动：土-结构相互作用的简化方法

> **一句话定位**：用一根**串联耦合**的「广义三明治梁」把高层结构简化为一维连续杆，用 4 个刚度参数同时刻画**整体弯曲/整体剪切/局部弯曲/局部剪切**四种机理，并在杆底挂**平动弹簧 $K_T$ + 转动弹簧 $K_R$** 把 [[土-结构相互作用]] 折进自由振动；闭式解（均匀结构）+ 传递矩阵法（变截面）求第一周期，对 1560 个算例与 FEM 对照，误差始终落在**安全侧**。是 23 篇兄弟里少见的**纯解析/简化抗震**路线，与重型三维 SSI（如 [[Scarfone2020R10]]、[[Heshmat2024R17]]）形成方法学对照。

## 研究对象
高层结构（含带大长度剪力墙的侧向支撑体系）的**自由振动**——核心目标量是**第一自振周期 $T_1$**。作者指出经典连续模型（[[Euler-Bernoulli 梁]]、[[Timoshenko 梁]]、剪切梁、耦合弯剪模型、经典三明治梁）普遍忽略三件事：**墙体局部剪切（local shear）**、**转动惯量（rotational inertia）**、以及沿高度的几何/材料变化。当剪力墙长度可观时会出现经典模型无法表达的局部剪切场，导致工程不可接受的误差。本文用一个能"退化为各经典模型特例"的广义模型补齐这三项，并把土柔性作为简化 [[Winkler地基模型]] 纳入。属作者 Pinto (2023) 静力/屈曲广义三明治梁工作的**动力学续作**。

## 模型（连续杆 / 刚度 / 边界 / 求解）
- **连续模型本体**：经典三明治梁与一根**纯剪切梁串联**（serial coupling）构成复合梁；后者再与 Timoshenko 梁、局部弯曲梁并联。三个运动学场：横向位移 $u$、整体弯曲转角 $\theta$、局部弯曲转角 $\varphi$。
- **四刚度**（仅依赖几何与静力属性）：整体弯曲 $K_{b1}=\sum EA_ic_i^2$；整体剪切 $K_{s1}=(K_b^{-1}+K_{c,w}^{-1})^{-1}$（梁与柱墙抗弯刚度串联）；局部弯曲 $K_{b2}=r\sum EI_i$；局部剪切 $K_{s2}=\sum GA_i'$。串联耦合让"局部剪切主导"的极端工况也能被表达——这是相对"并联耦合或直接忽略局部剪切"的经典模型的关键差异。
- **土柔性（SSI 入口）**：杆底加**平动弹簧 $K_T$** 与**转动弹簧 $K_R$**；势能多出 $\tfrac12 K_T u^2(0,t)+\tfrac12 K_R\theta^2(0,t)$。边界条件 $u(0,t)=V(0,t)/K_T$、$\theta(0,t)=M_1(0,t)/K_R$，刚度→∞ 退回固定基底。**故意不设竖向弹簧**（论证竖向振动对横向行为影响极小）。
- **控制方程**：Hamilton 原理 → 三个高度耦合的平衡方程 → 分离变量 → 化为关于 $\phi(x)$ 的**六阶常微分方程**（式 20），可退化为 Chesnais (2011) 方程（忽略转动惯量）与经典三明治梁方程（再忽略 $K_{s2}$，即令 $K_{s2}=\infty$）。
- **求解（两条路）**：① 均匀结构用 **Laplace 变换**得位移与内力闭式解，代入边界条件解 $\omega$；② 变截面用**连续法 + 传递矩阵法**结合，传递矩阵保持固定 $6\times6$ 阶（与层数无关），免去经典法中对零矩阵求逆，大幅降算量。
- **验证对象**：以 FEM 为"精确解"基准；算例覆盖 Zalka (2020) 三跨耦合剪力墙、Pinto (2023) 对称支撑体系建筑；总计 **2725 个分析算例**，其中参数分析 12 个构型（CSW1–12，墙长 5.0/7.5/10/12.5 m）× 5 种模型设定 × 楼层数 = **1560** 例；土柔性段用 4 类场地（Hard rock / Dense / Stiff / Soft）。

## 核心发现（带定量数字）
1. **局部剪切是精度关键，远胜转动惯量**。三跨耦合剪力墙验证：经典连续模型误差随高度增大而减小、最大达 **-12.92%**（结构不安全侧）；本文模型最大误差仅 **-3.85%**，且楼层越少改善越明显（低层局部剪切主导）。
2. **转动惯量影响很小，忽略反而更准**。参数分析：不计转动惯量（$\theta=0,\varphi=0$）最大误差 **7.37%**，计入（$\theta\neq0,\varphi\neq0$）反升至 **11.20%**。这印证 Chesnais (2011) 不含转动惯量的选择，也复刻 [[Timoshenko 梁]] 中"局部剪切影响 ≫ 转动惯量"的已知规律。但计入转动惯量会**增大周期**→降低地震输入力，作者据此建议用"只增转动惯量"的耗能装置减震。
3. **本文模型误差落在安全侧（偏保守），经典模型落在不安全侧**。令 $K_{s2}=\infty$（退回经典三明治梁）最大误差冲到 **33.99%**，Zalka (2020) 经典法 **-35.48%**；即本文模型把误差**降低约 4.6 倍**。两案例（Case 1/2）中经典法误差 **-11.51% / -15.75%**（不安全侧），本文模型 **+5.85% / +5.57%**（安全侧）。
4. **土柔性把第一周期改变可达 ~40%，且趋于"高度无关"**。相对 Hard rock 固定基底：Dense / Stiff / Soft 三类场地的第一周期变化最大值分别为 **14.44% / 23.83% / 39.76%**（平均绝对差 6.49% / 12.50% / 23.64%）。规律：**低层建筑变化为负**（柔性基底周期反小于固定基底），随楼层增加变为正，并**渐近稳定到与楼层数无关**的定值；土越软该正负切换越剧烈（Soft 约 -39.76% ↔ +38.93%）。

## 评注（客观）
- **方法学定位**：这是一条"轻量解析/降阶"路线——把整栋高层压成一根 6 阶 ODE 的连续杆，几秒得第一周期，与 23 篇兄弟中重型三维 FE/有限差分 SSI（[[Scarfone2020R10]]、[[Heshmat2024R17]]、[[Kanellopoulos2024R31]]）形成鲜明对照。代价是**线弹性 + 周期性假设**（要求 ≥5 层、墙内仅面内力、连梁不可伸长、楼板面内刚性），不适用强非线性/损伤工况。
- **SSI 简化层级**：土被压缩为底部两根弹簧（$K_T,K_R$，借用 Huergo & Hernández 2019 的参数表），属最简的"无质量弹簧基底"——不含辐射阻尼、不含运动相互作用（kinematic interaction）、不含频率依赖阻抗。与 [[fema-p-2091]]/[[nistgcr12-917-21]] 的阻抗函数体系相比是高度简化版，但足以定性捕捉"柔性基底拉长周期"的主效应。
- **"安全侧"论证的价值**：作者反复强调误差符号——经典模型低估周期（高估刚度→高估地震力？实则其误差落在结构不安全侧），本文模型偏保守。对周期敏感的等效侧力法估算有实用意义。
- **可证伪性**：所有结论以 FEM 为基准而非实测/振动台，结论的"精确"是相对 FEM 而非真实结构；土弹簧刚度取自文献而非现场反演。
- **图表近似值警示**：Fig 7/8/10–18 的误差与周期变化曲线在源文档中为 OCR 近似值（带 ~ 号），本文引用的关键定量数字优先取自正文与 Table 2–10 的精确百分比。
