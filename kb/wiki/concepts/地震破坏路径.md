# 地震破坏路径 (Seismic Failure Path)

结构在地震全过程中各部位**先后进入破坏的次序**。原多用于地上结构，[[邱大鹏]] 系列将其系统引入地下大型框架（[[ULSFS]]）与 [[SIUS]]。

## 本库收录的典型路径
- **ULSFS**（[[Qiu2024R04]]）：**外→内、下→上**；外侧 2nd/3rd 梁（配薄内楼板、刚度低）最先受弯屈服。
- **SIUS 垂直入射 0°**（[[Qiu2025R05]]）：地上底部梁 → 地下底部梁 → 地下顶部外侧梁 → 地上底部柱 → 地下底部柱 → 地上中部梁柱（**梁先于柱**）。
- **SIUS 斜入射 30°**（[[Qiu2025R05]]）：转为**地下优先**（地下底部梁 → 地下顶部外侧梁 → 地下底部柱 → …）。

"梁弱柱强"设计使梁先耗能、延缓整体失效。基于破坏路径的损伤评估法见 [[Qiu2024R04]]。
