> GFE 前处理各模块通过「几何集 gset / 表面集 surface / 节点集 nset」互相引用的规则。建模顺序或集合类型错了就报 MISS_REFERENCE。权威表（含施工步生死单元范例）见 raw [[GFE-CmdKB-objref.src]]。

## 引用规则（谁引用谁、用什么集）
- **截面属性** → 几何集（elset_name）+ 材料名（mat_name）
- **边界条件 / 约束** → 节点集/节点 ID/几何集均可（凡可选节点/单元集处都可直接给几何集；案例工作流惯用表面集）；压力 → 表面集；集中力 → 节点集；体力 / 惯性力 → 几何集 / 整模型
- **[[绑定约束]] tie** → 表面集对（first_surf 主 / second_surf 从）；推荐用「搜索接触」自动生成
- **[[通用接触]]** → 作用区域为两个表面集或 all exterior（*Contact Inclusions）；面面接触 → 表面对
- **[[嵌入区域]] embed** → embedded_names（嵌入体的几何集/单元集列表）+ host_name（主域几何集/单元集）
- **[[粘弹性人工边界]] artbc** → 表面集（土体外边界面）
- **工况 case** → 分析步名 + BC 名 + [[单元生死]]（set_elemAdd / set_elemDel 引用**几何集名**）+ 输出请求名

## 常见错误（→ 正确）
- 多几何体时截面直接用几何名 → 应建**几何集**（单几何体可直接用几何名，GFE 自动视为集合）
- Tie 引用几何集 → 应引用**表面集对**（需面正反属性）
- Model Change 引用几何对象名/单元编号 → 应引用**几何集名**（GUI 亦接受单元集）
