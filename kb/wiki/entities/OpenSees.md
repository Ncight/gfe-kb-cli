# OpenSees
开源地震工程有限元框架（Open System for Earthquake Engineering Simulation），由美国 UC Berkeley / PEER 中心主导开发，Tcl/Python 脚本驱动，强项在结构与岩土非线性时程、纤维梁柱、多面塑性土本构与 u-p 两相介质，是学界做 SSI / 桩-土 / 液化数值研究的主力平台（无内置 GUI，常配 GiD、HyperMesh 等可视化）。
