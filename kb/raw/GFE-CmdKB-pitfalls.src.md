# GFE 易错清单 —— LLM 驱动 GFE/PrePo 自动建模专用（2026-06-10 二审合成版）

## 0. 使用说明

本清单面向**任何用代码/命令流/GUI 自动化驱动 GFE（广州颖力岩土有限元）建模的 LLM**，尤其是推理能力弱于 Fable 的模型。三个数据源：①二审审计实证（Opus 实际犯过并已修正的 328 条错误，D:\GFE\GFE_KB\_audit\phase3\变更总表.md）；②用户机器实测坑（标 [实测]）；③8 本官方手册的 ~700 条反直觉点按杀伤力筛选。**建议把本文件整体作为 system prompt 附录注入**；至少在写任何 GFE 命令流/INP/操作序列前，按当前工作域（几何/材料/网格/荷载/工况/SSI/IO/后处理）查对应章节，并通读「红线十诫」与「Opus 实证错误模式」两节——后者描述的是 LLM 自身的系统性失败模式，不是软件知识。每条带出处锚点（手册§ / dump / 实测 / 变更总表 F-N），存疑时回真值源，**不要凭 Abaqus 印象补全**。

## 1. 红线十诫（最致命：崩溃 / 差数量级 / 静默失效）

1. **显式动力 \*Damping 的 β 必须=0**——非零触发质量缩放级联+CDP PD Failed 刷屏；隐式 β 合法（官方算例 0.0001~0.01），册间差异本身是坑。[实测 project_explicit_beta_damping；Implicit②#47]
2. **\*Wave/三向幅值点数必须一致**——Amp-X/Y/Z 对数不等 → 0xC0000374 堆损坏崩溃，跑前统一长度。[实测 project_gfe_wave_buffer_bug]
3. **BC/荷载/输出建好≠生效，必须挂入工况(Case)对应分析步**——GFE 比 Abaqus 多一层 Case，漏建/漏挂/挂错步全部静默不生效。（Cases②#59/#61）
4. **valid_dof 位语义与 Abaqus 相反：1=开放、0=限制**——写 7 想全约束实为放开三平动；全约束=0。（Cmd②#15；对象引用总览报批 F-2）
5. **非线性本构全部锁死显式动力**——静力/隐式下塑性参数静默退化为线弹性，不报错。（Explicit②#9；UG②#24）
6. **调幅/惯性力峰值单位是 m/s²**——案例 2.2=2.2 m/s²≈0.22g；按 gal 填 220 差百倍。（Cases②#87；Iso②#12）
7. **HSS/MC 摩擦角、剪胀角按弧度输入**——φ=3.14×40/180；直接填 40 差 57 倍。（Implicit②#10）
8. **场地反应分析必须跑两遍**——首遍只生成瑞利阻尼并回填材料，回填后必须重算才是最终结果。（FAQ②#30，§3.22）
9. **只交 inp 不带 inpx 会静默丢掉人工边界/地震载荷/工况/包络输出/构件偏心**——批处理提交尤其危险。（FAQ②#45；UG②#129）
10. **GFE 内部无量纲、不校验单位制**——逐案例自洽（m-t-kPa 或 mm-t-MPa 按 density/E 反推）；SPH 强制 SI(m)，CONWEP 强制 t-m-kPa，套错差数量级。[实测；Cases②#2/#101/#109]
11. **版本代际陷阱（双代际并存）：本机两套安装——D:\GFE PrePo=v2.15 代际（2025.07，运行中），E:\GFE2026=v3.x 完整安装（本机已装）**——v3.x 新 API（convert_to_davidenkov/revert_davidenkov/compute_era/build_non_uniform_soil/build_tunnel_shape/import_yjk/import_mat/find_near_node/copy_mesh/train_load 等）在 E:\GFE2026 可用、在 D:\GFE v2.15 不存在（v2.15 geotool 仅 10 个形状感知函数；GFE.soil 仅 box_builder/data_builder）。**工况配置两形态随版本**：v3.x=映射属性字典 `obj.bcs['步']=[...]`（可读）；v2.15=8 个 set_*（只写）——v3.2.2 release note"改为赋值形式"即指改成映射属性，KB 此前解读方向反了。写码前先探版本（hasattr/路径），跨代际脚本不可混写。[实测 2026-06-10 _audit/phase7/实测裁决.md；GFE2026 官方 py 实证 2026-06-11，raw 快照 GFE2026-CM-*]

## 2. 几何建模

### [高危] 布尔裁剪后只是贴合，不传力
- **错误示范**: 裁剪出基坑/孔洞后默认两体连成整体。
- **正确做法**: 必须再布尔合并（共节点）或绑定约束才传力。
- **出处**: Cases②#8，§1.2.1(7)/§2.2.3
### [高危] 先赋截面属性、再布尔合并，且勾"替换原始图形的集合"
- **错误示范**: 先合并再统一赋材料。
- **正确做法**: 异材料几何逐一赋属性→合并时勾替换才继承；勿把未赋材料的草图形状一起合并。
- **出处**: Cases②#9/#10，§2.2.3 注意3
### [高危] 布尔三分语义不同且选择有方向性
- **错误示范**: 把"分割"当"裁剪"用；先选工具体。
- **正确做法**: 合并=融合共节点/分割=印刻保留两侧/裁剪=删重叠体积；先选被裁体（土）→再选工具体。
- **出处**: Cases②#11/#12，§6.4.4/§6.5.3
### [高危] 分割/裁剪默认不勾"替换原始图形"→双份几何残留
- **错误示范**: 用默认选项裁剪后直接划网格。
- **正确做法**: 勾"替换"就地替换并保住材料；不勾则旧几何残留树中极易误选。
- **出处**: Cases②#13；FAQ②#14
### [红线] split() 后子实体父归属跨会话非确定，建集禁依赖 children 顺序/父 id/shape id
- **错误示范**: split 后按 children 下标或父 shape id 给子实体建集合/赋材料。
- **正确做法**: OCC 布尔排序不稳，官方 py 自注"每次打开 gfe 数据不一致"；建集只用 centre_of_mass 形心判定子实体身份，脚本重放/换会话才稳定。
- **出处**: GFE2026-CM-ch07 官方 py 自注 [官方实证 v3.x 2026-06-11]
### [提示] 共面/贴面布尔是隐性失败源，故意留间隙
- **正确做法**: 切割面先外移（如 5m）、拉伸长度超出被裁体（310>300）。
- **出处**: Cases②#14，§15.8.1(3)
### [提示] BOPAlgo 自相交告警刷屏≠失败
- **正确做法**: `BOPAlgo_Alert...` 海量刷屏可忽略；`Boolean Operation failed!` 才是真失败。
- **出处**: Cases②#15
### [高危] 草图遗留面/隐形图元必须手动删除
- **错误示范**: 拉伸完成后不管草图面。
- **正确做法**: 拉伸后草图面仍留树中须删，否则污染框选/布尔；"Failed to fill area" 先查隐形点线。
- **出处**: Cases②#18/#22
### [提示] 草图平面方向在画前设定，画完改无效；落位由"偏移"决定
- **出处**: Cases②#19/#20
### [提示] 阵列数量含原件、角度顺时针；"圆阵列"才是旋转复制，"旋转"只移动原件
- **出处**: Cases②#16/#17，§1.2.1(2)③
### [高危] 埋深靠移土不靠移结构
- **错误示范**: 把结构往下移到埋深。
- **正确做法**: 结构不动，土体形心对齐后整体上移（如 (0,0,4)）。
- **出处**: Cases②#28，§10.7/§11.8
### [提示] 几何修复（B 样条化）使集合失效；删几何集不连带失效引用方
- **正确做法**: 修复放建集之前；删集后手动"检查引用关系"。
- **出处**: UG②#20/#21
### [提示] 几何体删除后 id 不重排、不回收
- **正确做法**: 按原始编号定位；不能假设 id 连续。
- **出处**: FAQ②#34；Cmd②#22

## 3. 材料与本构

### [红线] 单元-材料配对是窄白名单，不是任意组合
- **错误示范**: 给壳配 Davidenkov/MC、给梁配超弹。
- **正确做法**: 纤维梁=线弹性/弹塑性/一维塑性损伤；平面应变=线弹性/MC/CDP/Davidenkov；壳=线弹性/弹塑性/CDP；实体=+MC/Davidenkov；Davidenkov 仅显式+三维实体。
- **出处**: Explicit②#8；Implicit②#27；UG②#25
### [高危] "一维塑性损伤本构"与 CDP 是两种材料
- **正确做法**: 一维版梁专用（抗压/抗拉强度+极限应变+软化系数）；多维 CDP 配壳/实体，参数集完全不同。
- **出处**: Explicit②#10
### [红线] HSS 必须前置 *Mohr Coulomb，标识位 3
- **错误示范**: 单独写 HSS UserMaterial。
- **正确做法**: *User Material 标识位 3 之前必须有 *Mohr Coulomb（手册加粗警告）；仅三维实体。
- **出处**: UG②#143；变更总表 CON HSS F-1/F-3
### [红线] *User Material 第一位是本构标识位不是物理参数
- **正确做法**: 1=一维混凝土,2=Davidenkov,3=HSS,4=南水,5=钢筋滞回,6=JH-2,7=E-v,8=E-B；参数从第二位起；constants 按浮点个数。
- **出处**: UG②#142，附录二(2)
### [红线] Davidenkov 的 *Depvar=29 个 SDV
- **正确做法**: 行写 `     29,`（前导空格原样），*Depvar 位于 *User Material 之前。
- **出处**: [实测] E:/论文数据源/2.5 Davidenkov本构
### [高危] CDP 两种计数语义并存+第 4 参是摆设+率效应仅 INPX
- **正确做法**: n_plasticity 数浮点个数，n_comp_harden 等数"组数"（21=42 浮点）；单行 5 参的第 4 位未用仍须占位；Johnson-Cook 率效应只能前处理设、落 INPX，手改 INP 加不上。
- **出处**: Cmd②#37/#39；UG②#137/#138
### [高危] 损伤恢复因子默认值相反
- **正确做法**: *Concrete Compression Damage 的 tension recovery 默认 0；*Concrete Tension Damage 的 compression recovery 默认 1。
- **出处**: UG②#139
### [高危] *Mohr Coulomb 数据行只读前两位
- **错误示范**: 按 Abaqus 填第三参数。
- **正确做法**: MC 与 MC Hardening 均仅前两位生效。
- **出处**: UG②#136
### [高危] INP 本构互斥五选一，*Density 永远必要
- **正确做法**: *Elastic / +*Plastic / +*Mohr Coulomb / +*CDP / *HyperElastic 互斥；不可叠加。
- **出处**: UG②#134
### [高危] 摩尔库伦是独立条目，不属 Drucker-Prager 家族
- **错误示范**: 把 MC 写成"DP 家族成员"并按 DP 屈服面填参。
- **正确做法**: GFE 中 MC 与 DP 并列；DP 是否支持四册手册零记载（待证）。
- **出处**: 变更总表 SUM Cases-06 F-2、CON 本构模型 F-1
### [红线] 官方弹塑性哲学：土不建 Davidenkov，弹塑性只落结构
- **错误示范**: 弹塑性 SSI 把土也转 Davidenkov（convert_to_davidenkov）。
- **正确做法**: 官方 py 全程土用弹性 gmat + UseEERAMat 等效线性（INP 导出自动生成 `<材料名>-EERA` 派生材料、逐层写 *Damping Alpha，Beta=0）；弹塑性只落结构——convert_material(1)+convert_reinforce(1)（此二 API 官方 py 实证存在）。convert_to_davidenkov 官方 py 也不用，GUI-only 坐实。
- **出处**: GFE2026-CM 各章官方 py [官方实证 v3.x 2026-06-11]
### [提示] 土体材料转换是"就地追加"非新建；重复转换返回 False
- **正确做法**: 转 Davidenkov 后同名材料多出参数块，有"还原"按钮；convert_material 重复调用返回 False 不一定是失败。
- **出处**: Cases②#39；Cmd②#65
### [提示] 弹性材料有"模量时间尺度"字段（默认"长期"）；elastic 必须显式 type=0
- **出处**: Cases②#37；Cmd②#41
### [提示] 一维非线性本构泊松比锁死 0.2；剪切失效用负数关闭（-1,-1）而非省略
- **出处**: UG②#27；Cmd②#40
### [高危] 南水模型 10/11 参数双口径+参数异名
- **正确做法**: 正文 10 参，INP 附录二 11 参（含 pa），脚本按 11 参写；表头 G_D=Cd、r_d=Rd，别名 NHRI。
- **出处**: UG⑤待仲裁#1；Implicit②#25
### [高危] 材料表倍率/表头单位会错，差 10³~10⁷
- **错误示范**: 照抄数字不看表头倍率；信"屈服强度(MPa)"表头。
- **正确做法**: 相邻表 E 单位 10⁷kPa vs 10⁴kPa/Pa 并存；某表头 MPa 实为 kPa（差 1000 倍）；粘聚力一处以 Pa 计。逐表核对倍率。
- **出处**: Explicit②#67/#69；Implicit②#9

## 4. 网格

### [红线] 显式步长=质量缩放"目标时间增量"，唯一入口
- **错误示范**: 以为按 CFL 自动调步长；在 *Dynamic 第一位设步长。
- **正确做法**: 未设默认 5e-5；案例域 2e-4~3e-5；INP 层 *Dynamic 只读第二位（总时长），步长看 *Variable Mass Scaling；GFE 同时锁死步长为最小步长（Abaqus 可上浮）。设大失真不报错。
- **出处**: Cases②#67；UG②#43/#164/#165
### [高危] 稳定步长 1e-8 量级=畸形单元报警
- **正确做法**: 跑 1s 短动力后在 *.feasta 搜 "mass" 诊断；质量缩放控制在 10% 内。
- **出处**: FAQ②#21，§3.1-5/6
### [红线] 节点集/单元集随网格重划静默失效
- **正确做法**: 网格未定稿先用几何集；单元集（复制网格产物）网格定稿后再生成。
- **出处**: Cases②#29；UG②#19
### [高危] 土体网格尺寸 ≤ 一维土层"计算"给的建议值（实证=Vs/50）
- **错误示范**: 凭经验拍网格尺寸。
- **出处**: Cases②#51；FAQ②#23；变更总表 SUM Cases-04 F-6
### [高危] 重剖须先手动删网格；全局网格参数是剖分时刻快照
- **正确做法**: 已有网格部件被静默跳过；换部件忘改参数即沿用上次设置。
- **出处**: UG②#17/#18；Cases②#45
### [高危] 复制网格三连坑
- **正确做法**: 产物名自动加 "-1" 后缀（填的是前缀）；类型不匹配到"确定"才报错；共节点必须显式勾选；生成的网格目前无法删除。
- **出处**: Cases②#43/#44；UG②#22
### [高危] 平面内嵌线不自动生成梁单元
- **正确做法**: 必须"复制网格"后补 B31 单元集。
- **出处**: Cases②#42，§3.1.5
### [提示] sweep ratio 是累计占比：递增、(0,1]、末项必须为 1
- **出处**: Cmd②#68；UG②#23
### [提示] 网格选项字典勿省键；geom_to_type 用固定模板
- **正确做法**: 三处官方示例均整套显式给全 {0:0,1:5,2:9,3:14,4:17,5:19,6:20,7:22,8:17}（键 4/8 同映射 17 非笔误）；单元形状切换会联动 2D/3D 算法默认值。
- **出处**: Cmd②#72/#73；Cases②#132
### [提示] 默认单元混搭：六面体 C3D8R（缩减）、四面体 C3D4（完全积分）；S3R/S4R 为默认壳
- **出处**: Cases②#48；UG③网格表

## 5. 边界与荷载

### [红线] 自重不叫重力，叫"惯性力"，g 取 -9.8 不是 -9.81
- **正确做法**: 类型=惯性力+分量3 手输负值，区域默认整个模型；对账勿"纠正"为 9.81；命令流走 boundary type=7。
- **出处**: Cases②#3/#4；Cmd②#106
### [红线] 边界挂 Initial 步、荷载/输出挂计算步——但非线性分析 initial 步禁放土体支座边界
- **正确做法**: 常规：BC→Initial、荷载→分析步；非线性 SSI：土底固定+四周法向约束放静力步、人工边界放动力步。
- **出处**: Cases②#54；FAQ②#41
### [红线] 重力必须施加于"整个模型"
- **错误示范**: 只给结构加重力。
- **正确做法**: 加载对象错误是非线性结果异常常见原因；覆土重量已由土密度+g 自动形成，再手动加附加质量=双计（轴压比巨大、损伤异常）。
- **出处**: FAQ②#42/#39
### [高危] 幅值函数是乘子非归一化；幅值函数2 仅频响有效
- **正确做法**: 最终值=荷载数值×幅值，表值可负可超 1；"幅值函数2"是复数虚部专用，其他分析填了静默无效；GUI 名叫"缩放函数"。
- **出处**: UG②#146；Cases②#56；Implicit②#7
### [高危] 压力荷载吃表面集且表面有正反
- **正确做法**: 先建表面集（实体默认外表面；二维面须选"品红"定向），方向错则压力/人工边界反向；INP 层 *Surface 仅支持 element。
- **出处**: Cases②#32/#55；UG②#132
### [高危] 位移/初速度边界不支持空间分布；隐式动力不支持速度/加速度边界
- **正确做法**: 空间变化强制位移拆集合；隐式动力用"位移"+幅值函数模拟。
- **出处**: Explicit②#37；UG②#30
### [高危] 荷载作用域硬限定
- **正确做法**: 压力仅 Surface；非结构质量仅节点集（且须配惯性力、放 Initial 步）；惯性力仅整体模型；集中力仅点集/节点集。
- **出处**: Explicit②#39；UG②#37
### [提示] "法向约束"不是独立 BC 类型
- **正确做法**: 用"位移/转动位移"勾单自由度（U1=0 或 U2=0）实现。
- **出处**: Cases②#58

## 14. Opus 实证错误模式（LLM 自身的系统性失败模式，源自 328 条已修正错误）

### a. 缺页/时代过时论断照写
- **描述**: 真值源缺页或基于旧版时，把"手册没讲 X / X 是隐藏 API"写成定论，实际新版已文档化。
- **实例**: "draft 模块手册无文档"——v2026 第 4 章整章文档化；"CDP 类隐藏"——§2.6.6 当时即有；geotool 3 函数"隐藏"实有文档。
- **防错规则**: 写"手册没有/隐藏"前必须 grep 最新版手册全文；版本敏感论断必须带版本号限定。
### b. 数字误植/易位
- **描述**: 转录数字时位易位、量级漂移，且复述时自信不减。
- **实例**: 土层合计 74.75↔75.85 m；"45 万节点"实为 189446（约 19 万）；para_int[44] 实为 43 元。
- **防错规则**: 关键数字（求和/计数/数组长度）必须程序复算，不靠目测转录；引用时保留原始出处行号供回查。
### c. 虚构不存在的 API/命令
- **描述**: 按"应该有"的直觉发明接口。
- **实例**: 命令流"PKPM 导入"（GUI 专有）；`gfe-daemon-dat` 连写（实为 `gfe -daemon -dat`）；`writer().set_case().perform()` 链式（set_case 返回 None）；"GFEXG.exe -db" 提交语法（出处缺失待实测）。
- **防错规则**: 任何 API 名/CLI 语法必须有 spec dump 或手册原文锚点；无锚点一律标"待实测"，禁止默写。
### d. 自动"纠正"原始数据的拼写/符号
- **描述**: 把真值源里"看起来像错"的内容顺手改对，导致与 .pre/dump 对不上。
- **实例**: Propery-tunnel 被改成 Property-（实际就少个 t）；OCR 顶底深度 "-12" 被保留负号（实为 12 非负）；-9.8 被"纠正"为 -9.81。
- **防错规则**: 名称/拼写以 dump 为准逐字照录，可加注"原拼写如此"；OCR 符号疑点标注待仲裁而非擅改。
### e. 把匿名对标软件写实
- **描述**: 手册全程用"软件A/S/Z"匿名，LLM 自作主张写成 ABAQUS/PLAXIS。
- **实例**: ABAQUS.md 标题把"软件A"写实，违反双手册明文匿名约束。
- **防错规则**: 衍生文档沿用匿名口径，最多注"社区共识推断"。
### f. 适用域以偏概全
- **描述**: 把单场景规则写成全局规则，或反向把全局写成单场景。
- **实例**: "β=0 恒定"（实仅显式）；"全局结构阻尼仅直接法频响"（实四类可用）；调幅 2.2 当通则（实为 ch10 案例值）；"几何只能名级感知"（shape 级可走 geotool）。
- **防错规则**: 每条规则写明适用域（求解器制式/分析类型/版本）；遇"恒/只/全部/必须"字眼自查反例。
### g. Abaqus 印象迁移
- **描述**: 用 Abaqus 语义脑补 GFE 行为。
- **实例**: valid_dof 1=约束（GFE 相反）；通用接触"全局自动搜索免枚举"（无真值支持）；质量缩放只缩质量（GFE 还锁步长）；按 Amplitude/CTF/ALLVD/SDV_Damage 找功能与变量（GFE 名全不同）。

### h. 两套同名概念混淆
- **描述**: 同名异义对象被当成一个。
- **实例**: 壳 SM1/SM2 下标轴交叉 vs 梁约定；同册两个"表 5.2"；EERA/ERA 双名；黏弹性边界 A/B vs Davidenkov A/B；三个同名"场输出"入口；"一维土层计算" vs "场地反应计算"。
